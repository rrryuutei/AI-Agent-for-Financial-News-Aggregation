What's Under the Hood?

GPT-4 does not know anything after Sept. 2021.
Our AI agent is able to utilize external tools like APIs to retrieve what it needs. In this way we inject the most recent news and data into the model. Here is how it works:

  1. Planning: Determine what are the most relevant stocks to explore.
  2. Retrieving: Use API to retrieve the most recent data and news about the stocks.
  3. Analyzing: Analyz the recent data and news, retrospect on its initial planning and retrieve more data to reflect the most recent trend.
  4. Summarizing: A well-written answer is summarized by the AI agent based on all the relevant news and data, with grounding information provided for the user.

Technical Details

  1. Skills: Python, Flask, React.js, Vite.js, Node. js, Websocket
  3. Back-end:
    ·Integrating Open API and Yahoo Finance API to retrieve news and stock market data, followed by summarization and categorization
    ·Employing Flask for Cross-Origin Resource Sharing (CORS) configuration to ensure that front-end applications can securely request resources from various origins.
  4. Front-end:
    ·Async Requests & Error Handling:Concurrently fetch data in React, tracking progress and handling errors gracefully with state management
    ·Event Listening & State Updates: Implement dynamic content visibility in React through wheel event listening and responsive state updates
